Como Trabajar con Listar

Definicion

Las listas en HTML sirven para contenido. En funcion del tipo de contenido de nuestra lista tenemos tres tipos de listas:

Cada elemento de las listas es representado con la etiqueta "li", tanto el las "ul" como en las "ol".

Mas Informacion

ul : Unordered List o (Lista Desordenada)

ul -> unordered list : Se utiliza cuando el orden de la lista no influye (Lista de Compra).

Mas Informacion

ol : Ordered List o (Lista Ordenada)

ol -> ordered list : Se utiliza para cuando el orden de los elementos es importante(Un top 10).

Mas Informacion

dl : Definition List o (Lista de Definiciones)

dl -> definition list : Se utiliza para hacer una lista de definiciones (diccionario).

Cada elemento de una lista de definicion lleva dos etiquetas:
dt -> Definition term : el termino que vamos a definir.
dd -> Definition description : la descripcion del termino.
