Elementos en Linea

los elementos de linea a diferencia de los elementos de bloque no ocupan todo el ancho de la web solo ocupan el espacio de su contenido dado asi que si ponemos otro elemento o etiqueta acontinuacion se posiciona exactamente a su lado

Ejemplos de Etiquetas que son elementos de Linea

em --> Enfasis

strong --> Mas enfasis

small --> Menos enfasis que el texto normal

br --> Forzar un salto de linea

wbr --> Salto de linea si Hiciera falta

time --> Se usa para representar un contenido de hora y fecha

i --> italic

b --> bold

u --> underline
