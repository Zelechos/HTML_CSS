Audios en HTML

Para que se muestre el el reproductor del audio se tiene que hacer los siguiente
<audio src="aqui va la ruta" controls>
agregamos el atributo controls para que se muestre los controles de reproductor y poder reproducir nuestro audio.

Tambien existe el atributo "autoplay" para que la musica se reproduzca automaticamente al recargar la pagina.
Tambien el atributo "muted" que sirve para silenciar el audio.
