Vocabulario Web Basico

Palabras Basicas que debes saber para empezar en el Mundo de Programacion Web

IP o Direccion IP

Es el Identificador numerico de una Pagina Web, es unico y representa la direccion donde esta el ordenador que contiene esa Pagina Web.

Mas Informacion

Dominio Web / URL(Uniform Resources Locator)

Es el nombre asociado a la IP que utilizamos para solicitar recursos, en nuestro caso un Sito Web.

Mas Informacion

DNS (Domain Name System)

Es un Servidor cuya principal funcion es traducir el nombre del dominio a su identificador unico.

Mas Informacion

Sitio Web

Es un conjunto de uno o varios recursos Web alojados en el mismo dominio.

Mas Informacion

Servidor Web

Es un Ordenador cuyo objetivo es servir recursos Web.

Mas Informacion

Hosting

Es el Almacenamiento del Servidor Web. El disco duro donde el servidor guarda los recursos.

Mas Informacion

Peticion

Es la accion de pedir recursos a un Servidor.
