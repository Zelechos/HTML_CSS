Etiquetas de Audio & Video

Ahora veremos como usar los atributos de la etiqueta de Audio en html

Etiqueta <audio src="">

<audio src=""> : El atributo src sirve para traer saber la ruta en donde esta nuestro audio que queremos reproducir.

Mas Informacion

Etiqueta <audio controls>

<audio controls> : El atributo controls sirve para habilitar los controles para poder reproducir el audio (SIN ESTE ATRIBUTO NO SE MUESTRA NADA EN LA WEB) .

Mas Informacion

Etiqueta <audio autoplay>

<audio autoplay> : El atributo autoplay sirve para que el audio se reproduzca automaticamente despues de actulizar la web.

Mas Informacion

Etiqueta <audio muted>

<audio muted> : El atributo muted sirve para silenciar el audio.

Mas Informacion

Etiqueta <audio loop>

<audio loop> : El atributo loop sirve para que el audio se vuelva a reproducir una vez termine el mismo.
